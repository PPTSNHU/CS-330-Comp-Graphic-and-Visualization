This project showcases skills I learned while in a class focused on the OpenGL graphics rendering API using C++.
It showcases skills focused on rendering meshes in a 3D environment and attenuating color, texture, light, and camera positioning
throughout the render. When creating this I applied industry best practices comment my code where I have edited the original 
framework to create a visual representation of the chicken coop in my backyard at home.
When I designed this render, I used the skills from class to the best of my knowledge to create complex shapes from multiple meshes
that incorporate some textures to give a life like look and feel to the scene. By understanding these methods and functions in the code 
I can create other renderings in the future that can be used to graphically display images, and with some more practice and time learning
animations, that can be implemented into applications. These skills are useful in graphical applications and the new knowledge I have gained in 
understanding OpenGL has given me new insight into how the images on computer screens are created by the GPU in tandem with the CPU.
In the future I hope to apply these skills and ideas to bolster my applications usability accessibility for end users.
